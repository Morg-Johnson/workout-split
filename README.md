## Weekly Workout Planner

Assign muscle groups or exercises to each day of the week. The app warns you if you train the same muscle group two days in a row.

### Run it

Open `index.html` in your browser.

If your browser blocks local storage or you prefer using a local server, run one of these from this folder:

```bash
python3 -m http.server 5173
```

Then visit:

`http://localhost:5173`
